== Description ==

Dieses Plugin erweitert die Funktionalität des "easyjob Webshop"-Plugin von protonic software um E-Mail-Optionen.
Es kann ein eigenes Prefix, der Name des Kunden sowie der angefragte Zeitraum mit ausgegeben werden.

== Installation ==

Die Installation dieses Plugins setzt das installierte und aktivierte "easyjob Webshop"-Plugin voraus.
Die Einstellungen können unter "Einstellungen" > "Easyjob Addons" festgelegt werden.
